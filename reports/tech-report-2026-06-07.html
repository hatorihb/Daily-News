npm / GitHub

IronWorm & Miasma 新亜種がnpmとGitHubを侵食 — Claude Code・VS Code も標的に
詳細を表示

6月初旬から続くサプライチェーン攻撃が拡大し、新たに「IronWorm」と命名されたRust製マルウェアが57件のnpmパッケージ・286以上の悪意あるバージョンに感染を広げた。preinstallフックで起動するELFバイナリがGitHub認証情報・クラウドクレデンシャル・CI/CDシークレットを窃取し、さらに自己複製して他パッケージへ感染を拡散する。6月5日にはMicrosoft Azure GitHubの73リポジトリが侵害されGitHubが一時アクセスを無効化した。Claude Code・Gemini CLI・Cursor・VS Codeなど主要AI開発ツールが起動時に悪意あるペイロードを実行するよう設定ファイルを改ざんする手口も確認されており、開発者は自身のnpm環境とGitHub Actionsの設定を即時点検することが推奨される。

CISA が Magento 向け RCE 脆弱性 CVE-2026-45247 を KEV カタログに追加
