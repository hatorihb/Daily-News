AWS CloudFront
重要度 高 2026-07-16
AWS CloudFront 障害——VPC Origins 利用顧客に 5xx エラーが多発、複数リージョンの Web サービスが一時停止
詳細を表示

Amazon Web Services の CDN サービス CloudFront が7月16日（PDT 午前1時45分、UTC 午前9時45分）ごろから障害を起こし、VPC Origins を使用している顧客を中心に 5xx エラーが多発した。世界複数リージョンにわたってウェブサービスやオンラインアプリケーションが一時的に停止し、利用企業のサービス品質に広範な影響を与えた。AWS は VPC 以外のオリジンタイプへの切り替えを暫定回避策として案内するとともに、根本原因の特定と修復作業を進めた。VPC Origins を使用しない顧客への影響はなかったとされる。CloudFront は世界中のウェブサービスが広く採用する大規模 CDN であり、今回の障害は AWS インフラ依存の高さを改めて示す事例となった。
