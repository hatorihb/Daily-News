AWS Step Functions に AgentCore 連携のエージェント推論ステップが追加（プレビュー）
詳細を表示

AWS Step Functions が Amazon Bedrock AgentCore のマネージドハーネスと連携し、ワークフロー内に AI エージェントの推論ステップを追加できるようになりました（プレビュー）。文書分類や非構造フォームからの情報抽出などをエージェントが自動化でき、複数エージェントの並列・逐次実行や重要アクション前の人間承認ステップの組み込みも可能です。実行履歴ではエージェントの入出力・トークン使用量・所要時間を Amazon CloudWatch 上で詳細確認でき、追加の統合料金はかかりません。対応リージョンは米国東部（バージニア北部）、米国西部（オレゴン）、欧州（フランクフルト）、アジア太平洋（シドニー）です。

Check Point VPN ゼロデイ（CVE-2026-50751 / CVSS 9.3）を Qilin ランサムウェアが1ヶ月以上悪用
詳細を表示

CVSSスコア9.3のクリティカルな脆弱性 CVE-2026-50751 が Check Point のリモートアクセス VPN・Mobile Access に存在することが6月8日に公表されました。廃止済みの IKEv1 プロトコルにおける証明書検証ロジックの欠陥を突き、未認証の攻撃者がパスワード認証を完全に回避して VPN セッションを確立できます。Qilin ランサムウェアグループの関連組織が5月7日から攻撃を開始しており、パッチ提供の約1ヶ月前から悪用されていたことが明らかになっています。AI 搭載の Spark ファイアウォールを含む複数製品向けホットフィックスがリリース済みで、IKEv1 設定を利用しているユーザーは即時適用が求められます。

Cisco は Catalyst SD-WAN Manager に深刻な特権昇格の脆弱性 CVE-2026-20245（CVSSスコア7.8）が存在し、実際に悪用されていると警告しました。コマンドラインインターフェースのユーザー入力検証の不備に起因し、netadmin 権限を持つ認証済み攻撃者が任意のコマンドを root として実行できます。エッジデバイスへの不正な設定変更が発生した被害事例も報告されており、オンプレミス・クラウドを含む全デプロイ形態が影響を受けます。修正パッチは現時点では提供されておらず、過去の SD-WAN 脆弱性（CVE-2026-20182 など）と連鎖させた攻撃にも警戒が必要です。
