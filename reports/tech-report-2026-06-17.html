Cisco / CVE-2026-20262
重要度 高 2026-06-16
Cisco Catalyst SD-WAN Manager のゼロデイ CVE-2026-20262 が実環境で悪用確認 — 緊急パッチを公開
詳細を表示

Cisco は6月15〜16日、Catalyst SD-WAN Manager のファイルアップロード処理における入力検証不備に起因するゼロデイ脆弱性 CVE-2026-20262（CVSS 6.5）のパッチを公開した。認証済みのリモート攻撃者が細工した HTTP リクエストをAPIエンドポイントに送信することでシステムファイルを書き換え、root 権限に昇格できる。オンプレミス・Cisco SD-WAN Cloud-Pro・FedRAMP を含む全デプロイ形態が対象であり、Cisco の社内テスト中に発見されたが6月時点で限定的な実環境悪用が確認されている。修正済みバージョン（20.9.9.2 / 20.12.7.2 / 20.15.4.5 / 20.15.5.3 / 20.18.3.1 / 26.1.1.2）への早急なアップグレードが必要である。

CISA / CVE-2026-54420
重要度 高 2026-06-16
CISA: LiteSpeed cPanel プラグインの権限昇格脆弱性 CVE-2026-54420 を KEV に追加 — 6/18 までに対応必須
詳細を表示

米サイバーセキュリティ・インフラセキュリティ庁（CISA）は6月16日、LiteSpeed cPanel / WHM プラグインに存在する権限昇格の脆弱性 CVE-2026-54420（CVSS 8.5）を「悪用が確認された脆弱性カタログ（KEV）」に追加した。FTP またはウェブシェルアクセスを持つ攻撃者がシンボリックリンクを悪用し、CloudLinux / CageFS 環境の共有ホスティングサーバー上で root 権限に昇格できる。連邦政府機関（FCEB）は6月18日までに LiteSpeed WHM Plugin v5.3.2.1（cPanel Plugin v2.4.8 以上）への更新が義務付けられており、共有ホスティング事業者やレンタルサーバー利用者も早急な確認と対応が求められる。同じ LiteSpeed プラグインには5月末にも別の CVE（CVE-2026-48172）が KEV に追加されており、継続的な脆弱性管理が課題となっている。
